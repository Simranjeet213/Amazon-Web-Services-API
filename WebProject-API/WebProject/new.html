Calculator

The following form will allow input in first number and second number, then choose options for operator like '+' for addition, '-' for subtraction, '*' for multiplication, '/' for division, '!' for factorial, '^' for power and '%' for modulus, and press calculate.
To find factorial, enter number in first number, then select '!' and press calculate.

OUTPUT with the use of AJAX
AJAX is used to connect HTML page from client end to server side where API is deployed through java Script, which can be possible with XMLHttpRequest. AJAX is used to load the specific content of HTML page instead of loading whole page. Html page does not need server to execute it directly run from web browser. So the origin is null. CORS is used to make request to server where the request is originated. To do so we have to add cors to Api which is deployed on server. CORS is part of the AJAX engine.
Version2 API is used to perfom operations and then ajax loads that data into html page.
